### Installation
This project requires the following external node modules:
- `socket.io`

To install the module, `cd` into the project directory and run
```
npm install
```

#### Launch
To run the server, `cd` into the project directory and run

```
node server.js
```

and visit the webpage at http://localhost:3000/chatClient.html.
