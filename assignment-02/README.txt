### Installation
No external `npm` modules are required for this project.

#### Launch
To run the server, `cd` into the project directory and run

```node server.js```

and visit the webpage at http://localhost:3000/index.html.

#### Testing
To test the app, search for a puzzle (ie. `puzzle1`, `puzzle2`, etc.) at the text prompt and click "Get Puzzle". The canvas will populate with scrambled puzzle. Next, put the pieces in the correct order and click "Solve Puzzle". If all the pieces are lined up, then it will output green text if the solution is correct, or red otherwise. For best results, leave as much space between lines as possible.
